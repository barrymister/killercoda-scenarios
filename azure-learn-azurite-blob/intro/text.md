# Azurite — the local Azure Storage emulator

**Azurite** implements the real Azure Blob/Queue/Table APIs locally, so the same
`az storage` commands you'd run against Azure work against it — for free, offline.
It uses a well-known dev account (`devstoreaccount1`) and connection string.
You'll create a container and upload a blob, exactly as in Domain 2.

Azurite (in Docker) and the Azure CLI are starting.
