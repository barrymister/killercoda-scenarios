# Bicep — Azure infrastructure as code

**Bicep** is Azure's language for describing resources declaratively. You write a
`.bicep` file, and `az bicep build` transpiles it to an ARM JSON template Azure
can deploy. It's the #1 IaC skill on AZ-104 (Domain 3), and it compiles
**completely offline** — no Azure subscription needed to practise authoring.

The Azure CLI and Bicep are installing for you. Give it a minute, then start.
