# Terraform with the azurerm provider

Terraform is the multi-cloud IaC tool; for Azure it uses the **azurerm** provider.
`terraform init` downloads the provider and `terraform validate` checks your HCL
is well-formed — both run **without** an Azure login, so you can practise
authoring for free. AZ-104 favours Bicep, but knowing Terraform is a real plus.

Terraform is installing now.
